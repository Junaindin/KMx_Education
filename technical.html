OS OptimizationWindows
Windows 11 Games සහ Softwares සඳහා උපරිම ලෙස සකසමු
ඔබේ පරිගණකයේ ක්‍රියාකාරීත්වය (Performance) මන්දගාමී නම්, Windows 11 මෙහෙයුම් පද්ධතිය ක්‍රමවත්ව Optimize කරගැනීම අත්‍යවශ්‍ය වේ. පළමුවෙන්ම අනවශ්‍ය Background Apps සහ Startup Programs අක්‍රීය (Disable) කරන්න. Settings -> Apps -> Startup වෙත ගොස් මෙය සිදුකළ හැක. මීට අමතරව Storage Sense සක්‍රීය කිරීමෙන් තාවකාලික ගොනු (Temporary Files) ස්වයංක්‍රීයව මැකී යයි. හොඳම ප්‍රතිඵල සඳහා Game Mode සක්‍රීය කර තබා ගන්න. එමඟින් පද්ධතියේ සම්පත් (CPU/RAM) ධාවනය වන ක්‍රීඩාව හෝ මෘදුකාංගය සඳහා උපරිමයෙන් වෙන්කර දෙනු ලබයි. අවසාන වශයෙන්, Advanced System Settings වෙත ගොස් Visual Effects "Adjust for best performance" ලෙස සැකසීමෙන් පැරණි පරිගණක පවා ඉතා වේගවත් කරගත හැක.
Terminal ArchitectureLinux
Cyber Security සඳහා Linux Terminal මූලික විධානයන්
Parrot OS හෝ Kali Linux වැනි පද්ධති පරිහරණය කිරීමේදී Terminal විධානයන් (Commands) පිළිබඳ අවබෝධය ඉතා වැදගත් වේ. මූලිකම විධානයක් වන `sudo apt update` මඟින් මුළු පද්ධතියේම Package Repository යාවත්කාලීන කරනු ලබයි. නව මෘදුකාංග ස්ථාපනය කිරීමට `sudo apt install [package_name]` භාවිතා කළ හැක. පද්ධතියේ දැනට ක්‍රියාත්මක වන ෆෝල්ඩරය වෙනස් කිරීමට `cd` විධානයත්, එහි ඇති ගොනු ලැයිස්තුව බලාගැනීමට `ls` විධානයත් භාවිත වේ. ජාලකරණ ගැටලු නිරාකරණය සඳහා (Network Troubleshooting) `ping` සහ `ifconfig` (නැතහොත් `ip a`) යන මෙවලම් උපකාරී වේ. මේවා නිවැරදිව හැදෑරීම ආරක්ෂිත ජාල පරිපාලනයට මඟ පාදයි.
